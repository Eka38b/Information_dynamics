# Utils

This directory contains utility scripts for analyzing and visualizing
results generated by the framework.

These tools are primarily used for:
- plotting temporal dynamics
- generating figures for analysis or publication
- inspecting node- and link-level information variables

---

## Main Script

### plot_results.py

This is the primary utility script for visualization.

It supports plotting:
- link-based quantities (e.g., MI, TE, rTE)
- node-based quantities (e.g., H)

---

## Basic Usage

Run from the repository root.

### Plot link dynamics

```bash
python -m Utils.plot_results \
    --dir <result_directory> \
    --link <node1> <node2>
```

Example:

```bash
python -m Utils.plot_results \
    --dir on_Model/004_ABN_for_GRN/Temporal_Results/ \
    --link B4 B5 --paper
```

---

### Plot node dynamics

```bash
python -m Utils.plot_results \
    --dir <result_directory> \
    --node <node>
```

Example:

```bash
python -m Utils.plot_results \
    --dir on_Model/004_ABN_for_GRN/Temporal_Results/ \
    --node B1 --paper
```

---

## Options

- `--paper`  
  Format plots for publication (clean layout, adjusted fonts, etc.)

- `--dir`  
  Path to the directory containing saved results

- `--link`  
  Specify two nodes defining a link

- `--node`  
  Specify a single node

---

## Output

- Figures are displayed and/or saved depending on script configuration
- Plots typically include:
  - MI
  - TE
  - rTE
  - H (for nodes)

---

## Notes

- Ensure the result directory exists before running the script
- Paths should be given relative to the repository root
- Plot appearance may depend on matplotlib configuration

---

## Role in the Framework

The `Utils` module provides the bridge between:
- raw simulation output
- interpretable results and figures

It is especially useful for:
- verifying behavior during development
- generating figures for manuscripts
