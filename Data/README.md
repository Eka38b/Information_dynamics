# Data Directory — Dataset v0.1.3

This directory contains the simulation outputs associated with the manuscript:

“Robustness of Information Circulation under Entropy Constraints”

---

## Overview

All data are generated by the simulation scripts in "on_Model/" and "on_Equations/".
They include time series of information-theoretic quantities and related variables used to produce the figures in the manuscript.

The data are provided to ensure full reproducibility without requiring recomputation.

---

## Directory Structure

The data are organized by model and experiment:

Data/
├── on_ModelXXX/
├── on_EquationsXXX/

- "on_ModelXXX/": Data generated from stochastic simulation models
- "on_EquationsXXX/": Data generated from direct evolution of information-theoretic variables

Each subdirectory corresponds to a specific simulation setup.

---

## Data Used in This Manuscript

The following directories are used to generate the results reported in the manuscript:

- on_Model015/
  → Figure 3 (Boolean probability update model)

- on_Equations005/
  → Oscillatory dynamics of two interacting cycles

- on_Equations001/
  → Small deviation from a stationary single cycle

- on_Model001/
  → Transient dynamics with an external information source

---

## Additional Data (Not Used in This Manuscript)

The following directory is included in the repository but is not used in the present manuscript:

- on_Model004/

This dataset is associated with a separate study and is retained for completeness of the repository.
It can be ignored when reproducing the results of this work.

---

## File Contents

Within each dataset directory:

Temporal_Results/
├── *_E_values.txt   # Control or interaction terms (E)
├── *_MI.txt         # Mutual information time series
├── *_TE.txt         # Transfer entropy time series
├── *_rTE.txt        # Reversed transfer entropy (if available)
└── Figure.png       # Generated figure (if applicable)

---

## Reproducing the Data

Each dataset can be regenerated using the corresponding script:

Dataset| Script

on_Model015| "python3 -m on_Model.015_Boolean_Probability_Update.main"

on_Equations005| "python3 -m on_Equations.005_Oscillatory_Two_Cycles.main"

on_Equations001| "python3 -m on_Equations.001_A_Single_Cycle.main"

on_Model001| "python3 -m on_Model.001_Toy_Model_A.main"
