# Data

This directory stores archived outputs generated by selected simulations in the repository. The files are provided for reproducibility, comparison with newly generated runs, and reuse in later analyses.

The `Data/` directory is not restricted to a single manuscript. Some subdirectories correspond to published or submitted studies, while others are retained as general simulation outputs from earlier examples.

---

## Organization

```text
Data/
├── on_Model001/
├── on_Model004/
├── on_Model015/
├── on_Equations001/
└── on_Equations005/
```

The naming convention links each archived dataset to the script family that produced it:

- `on_ModelXXX/` contains outputs from model-based simulations in `on_Model/`.
- `on_EquationsXXX/` contains outputs from direct information-equation simulations in `on_Equations/`.

For example, `Data/on_Model015/` corresponds to the simulation code in:

```text
on_Model/015_Boolean_Probability_Update/
```

and `Data/on_Equations005/` corresponds to:

```text
on_Equations/005_Oscillatory_Two_Cycles/
```

## Current Contents

### `on_Model001/`

Archived output from `on_Model/001_Toy_Model_A/`.

This example describes a cyclic model with an external information source and is useful for analyzing transient transfer-entropy responses under external input.

The datasets in this directory were primarily produced for the manuscript available in /Manuscripts/002.

### `on_Model004/`

Archived output from `on_Model/004_ABN_for_GRN/`.

This dataset is associated with an asynchronous Boolean-network example for a gene-regulatory-network setting. It is retained for completeness and may correspond to a separate study from the other archived outputs.

The datasets in this directory were primarily produced for the manuscript available in /Manuscripts/001.

### `on_Model015/`

Archived output from `on_Model/015_Boolean_Probability_Update/`.

This example uses a Boolean probabilistic update rule to study competition between internal and external information inputs at a merging node.

The datasets in this directory were primarily produced for the manuscript available in /Manuscripts/002.

### `on_Equations001/`

Archived output from `on_Equations/001_A_Single_Cycle/`.

This example studies direct information-variable evolution on a single cycle and is useful for checking near-stationary behavior under small deviations.

The datasets in this directory were primarily produced for the manuscript available in /Manuscripts/002.

### `on_Equations005/`

Archived output from `on_Equations/005_Oscillatory_Two_Cycles/`.

This example studies two interacting information cycles and includes oscillatory behavior of transfer entropy, reversed transfer entropy, and mutual information.

The datasets in this directory were primarily produced for the manuscript available in /Manuscripts/002.

## Regenerating Data

Run scripts from the repository root. Representative commands are:

```bash
python -m on_Model.001_Toy_Model_A.main
python -m on_Model.004_ABN_for_GRN.main
python -m on_Model.015_Boolean_Probability_Update.main
python -m on_Equations.001_A_Single_Cycle.main
python -m on_Equations.005_Oscillatory_Two_Cycles.main
```

Some scripts write new outputs to their own `Temporal_Results/` directories. Archived copies in `Data/` are kept separately so that existing results remain available for comparison.

---

## Plotting Archived Data

Use the top-level plotting utility:

```bash
python -m Utils.plot_results --dir <data_directory> --links <source target ...> --keys <quantity>
```

Example:

```bash
python -m Utils.plot_results \
  --dir Data/on_Equations001/ \
  --links A1 A2 A2 A3 A3 A4 A4 A5 A5 A6 A6 A7 A7 A8 A8 A1 \
  --keys MI
```

## Notes

- Archived data are kept as plain text for transparency and portability.
- New simulations may differ slightly if random seeds, parameters, package versions, or estimator settings are changed.
- When using data associated with a specific manuscript or release, cite the corresponding release DOI if available.
